Without a token you can see public repos only and are limited to 60 API requests per hour.

With a token you get your private repos and 5,000 requests per hour.

How to get one: Go to github.com/settings/tokens/new, select public_repo scope (or repo for private repos), generate, and paste it here.

Your token is stored in your browser session only (cleared when you close the tab) and sent exclusively to the GitHub API. No server, no tracking, no third parties.
